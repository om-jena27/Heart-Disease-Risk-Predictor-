# Heart Risk Predictor (AIML Project)

A machine learning web application built using Python and Streamlit to predict the likelihood of heart disease based on user health metrics. 

## Features
- **Machine Learning Model:** Uses `RandomForestClassifier` from Scikit-Learn.
- **Web Interface:** Interactive UI built with Streamlit.
- **Dynamic Training:** For the purpose of this demo, a synthetic dataset is generated and the model is trained automatically upon running the application.

## How to Run

1. Simply double-click on `start app.bat`. 
2. It will automatically check for Python, install the required libraries like `streamlit` and `scikit-learn`, and launch the application in your default web browser.

## Technologies Used
- **Python** (Core language)
- **Streamlit** (Frontend framework)
- **Scikit-Learn** (Building the Random Forest classification model)
- **Pandas & Numpy** (Data processing and generation)
